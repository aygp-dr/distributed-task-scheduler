# distributed-task-scheduler

Interactive task queue viewer with job control. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/distributed-task-scheduler
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
