#+TITLE: distributed-task-scheduler
#+AUTHOR: aygp-dr

Interactive task queue viewer with job control. Terminal UI built with [[https://github.com/charmbracelet/bubbletea][Bubble Tea]].

** Run

#+begin_src bash
go run .
#+end_src

** Build

#+begin_src bash
make build
./bin/distributed-task-scheduler
#+end_src

** Controls

- =j/k=: navigate
- =Enter=: select/details
- =q=: quit
